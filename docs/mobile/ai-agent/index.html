AI 对话 ​

AI 对话能力用于帮助您在手机上完成命令执行、问题排查和运维咨询。

当前主要包含 Chat 模式和 Term 模式，两者可通过输入框底部的模式选择器切换。

Chat 模式 ​

Chat 模式用于与 AI 协作完成分析、解释和命令建议。

使用：

  • 需要先登录账号
  • 需要在个人中心选择可用模型
  • 支持文字输入
  • 支持语音输入
  • AI 可以解析自然语言并生成可执行命令
  • 支持根据上下文持续协助排查问题

适合场景：

  • 询问 Linux 命令用法
  • 获取故障排查建议
  • 请求脚本编写辅助
  • 理解配置文件含义
  • 咨询运维最佳实践

Term 模式 ​

Term 模式用于直接执行命令，不经过 AI 处理。

使用：

  • 需要先连接服务器
  • 未连接时，页面头部会显示“点击连接主机”入口
  • 支持文字输入
  • 支持语音输入
  • 返回命令执行结果

适合场景：

  • 临时执行一条命令
  • 快速查看主机状态

典型使用流程 ​

  1. 打开 AI 对话页
  2. 先连接目标主机
  3. 选择 Chat 或 Term
  4. 输入问题、命令或通过语音发起请求
  5. 查看返回结果并继续追问

使用建议 ​

  • 需要解释、规划和排障时优先使用 Chat 模式
  • 登录后先在个人中心确认模型设置，避免进入 Chat 模式后无法使用
