# 移动端常见问题

## 服务器连接失败？

- 检查网络连接是否正常
- 关闭网络代理工具（代理可能干扰 SSH 连接）

## APP 切到后台连接断开？

- Android：将应用的省电策略设置为 `无限制`，可以减少后台断连

## 为什么会出现资产数据消失？

常见原因有两类：

- 数据隔离：不同账户使用独立的数据集，登出或切换账号后会看到不同的资产列表
- 应用卸载：卸载应用会同时删除本地数据

## 如何反馈问题？

你可以通过以下方式联系 Chaterm：

- 应用内反馈：`个人中心 > 使用与反馈 > 问题反馈`
- 邮件支持：`support@chaterm.ai`

## 支持哪些 AI 模型？如何切换？

登录后可在[个人中心 > AI 设置 > 模型选择](/docs/mobile/profile/#模型选择)中查看并切换。目前支持 Kimi2.5、GLM-5 等主流厂商的旗舰模型，不同订阅类型拥有的模型不同，以实际为准。

## 数据同步失败怎么办？

- 确认已登录账号（未登录时同步不可用）
- 检查网络连接是否正常
- 在[个人中心 > 数据管理](/docs/mobile/profile/#数据管理)确认同步开关已开启
- 如问题持续，可尝试重新登录后再同步

## 未登录可以使用哪些功能？

未登录时可以添加和管理本地资产、建立 SSH 连接。AI 对话功能和数据云端同步需要登录后才能使用。

## 排查建议

如果问题与连接或登录相关，请先确认：

- 应用版本是否为最新版
- 当前网络是否可访问目标主机
- 账号是否处于登录状态
- 模型和 AI 配置是否已正确设置
