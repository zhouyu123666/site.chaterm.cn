# 终端操作

终端是 Chaterm 的核心功能 -- 提供完整的 SSH 终端体验，支持分屏、搜索和键盘驱动的工作流。

## 终端连接

连接主机后，终端窗口会自动打开。终端支持：

- **全功能 SSH 终端**：支持所有标准命令和操作。
- **多标签管理**：可以同时运行多个会话，并在它们之间切换。
- **标签重命名**：右键点击标签即可重命名，方便快速识别。
- **标签克隆**：可以在同一主机上复制当前会话。

## 终端分屏

分屏功能可以让您并排或上下查看多个会话。每个分屏窗格都是独立的终端会话，可以连接到不同的主机或运行不同的命令。

### 分屏方法

1. 打开标签菜单栏或在终端内右键点击。
2. 选择 **向右分屏** 或 **向下分屏**。
3. 新的终端窗格会出现在选定的方向。
4. 每个窗格独立运行 -- 可以单独输入命令、调整大小或关闭。

### 分屏快捷操作

| 功能         | macOS         | Windows/Linux |
| ------------ | ------------- | ------------- |
| 向右分屏     | 通过标签菜单  | 通过标签菜单  |
| 向下分屏     | 通过标签菜单  | 通过标签菜单  |
| 克隆会话     | 通过标签菜单  | 通过标签菜单  |

> **提示：** 使用左右分屏可以在左侧查看日志、右侧执行命令。使用上下分屏可以在上方监控系统指标、下方进行操作。

## 终端搜索

在终端输出中搜索命令、错误信息或任何文本。

### 操作步骤

1. 按 `Cmd + F`（macOS）或 `Ctrl + F`（Windows/Linux）打开搜索栏。
2. 输入搜索关键词。
3. 匹配的文本会在终端输出中高亮显示。
4. 点击 **上一个** / **下一个**（或使用箭头按钮）在匹配项之间跳转。
5. 按 `Esc` 关闭搜索栏。

### 搜索快捷键

| 功能       | macOS         | Windows/Linux |
| ---------- | ------------- | ------------- |
| 打开搜索   | `Cmd + F`     | `Ctrl + F`    |
| 关闭搜索   | `Esc`         | `Esc`         |

## 右键菜单操作

在终端任意位置右键点击可打开上下文菜单。

### 剪贴板

| 功能     | macOS         | Windows/Linux | 描述               |
| -------- | ------------- | ------------- | ------------------ |
| **复制** | `Cmd + C`     | `Ctrl + C`    | 复制选中的文本     |
| **粘贴** | `Cmd + V`     | `Ctrl + V`    | 粘贴剪贴板内容     |

### 显示

| 功能         | macOS           | Windows/Linux   | 描述                       |
| ------------ | --------------- | --------------- | -------------------------- |
| **搜索**     | `Cmd + F`       | `Ctrl + F`      | 打开搜索栏                 |
| **清屏**     | 通过右键菜单    | 通过右键菜单    | 清除终端可见输出           |
| **字体大小** | 通过右键菜单    | 通过右键菜单    | 放大或缩小终端字体         |

### 连接

| 功能           | 描述                     |
| -------------- | ------------------------ |
| **断开连接**   | 断开当前 SSH 连接        |
| **新终端**     | 打开新的终端会话         |
| **关闭终端**   | 关闭当前终端会话         |

### 分屏

| 功能         | 描述               |
| ------------ | ------------------ |
| **向右分屏** | 创建水平分屏窗格   |
| **向下分屏** | 创建垂直分屏窗格   |

## 所有键盘快捷键

| 功能             | macOS         | Windows/Linux |
| ---------------- | ------------- | ------------- |
| 复制             | `Cmd + C`     | `Ctrl + C`    |
| 粘贴             | `Cmd + V`     | `Ctrl + V`    |
| 搜索             | `Cmd + F`     | `Ctrl + F`    |
| AI 命令对话框    | `Cmd + K`     | `Ctrl + K`    |
| AI 侧边栏       | `Cmd + L`     | `Ctrl + L`    |

完整快捷键列表请参阅 [快捷键设置](/docs/settings/shortcuts/)。
