# 个人信息

查看和管理你的账户详情、订阅等级和安全信息。

## 访问个人信息

点击左侧菜单栏中的 **User** 图标，打开个人信息页面。

## 账户信息

### 订阅等级

当前方案显示在个人信息页面顶部。Chaterm 提供四个等级：

| 等级 | 说明 |
|------|-------------|
| **FREE** | 基础功能访问，功能有限 |
| **LITE** | 面向个人用户的扩展功能 |
| **PRO** | 面向高级用户的完整功能 |
| **ENTERPRISE** | 团队管理和高级控制 |

### 基本信息

| 字段 | 说明 |
|-------|-------------|
| **姓名** | 你的真实姓名，显示在个人资料和共享空间中 |
| **用户名** | 用于登录的唯一标识符 |
| **邮箱地址** | 用于接收通知、验证码和账户恢复 |
| **手机号码** | 用于安全验证和短信通知 |

### 安全信息

| 字段 | 说明 |
|-------|-------------|
| **密码** | 账户登录密码，可在此页面重置 |
| **IP 地址** | 当前登录会话的网络地址 |
| **MAC 地址** | 当前使用设备的硬件地址 |
