# 修改密码

修改你的账户密码，保障 Chaterm 账户安全。

## 操作步骤

1. 点击左侧菜单栏中的 **User** 图标，打开个人信息页面。
2. 找到并点击**重置密码**按钮。
3. 在密码输入框中输入新密码。
4. 点击**确认**保存更改。

::: warning 密码要求
- 密码必须包含至少 **8 个字符**。
- 建议使用字母、数字和特殊字符的组合，以提高安全性。
- 请不要使用曾经使用过的密码。
:::
