# 账号管理

管理你的账户注册、认证方式和安全设置。

## 账户注册

### 邮箱注册

使用邮箱地址进行账户注册。系统会向你的收件箱发送验证码以确认身份。

### 手机号注册

使用手机号进行账户注册。系统会向你发送短信验证码以确认身份。

### 第三方认证

使用以下平台的现有账户快速登录：

| 平台   | 说明             |
| ------ | ---------------- |
| **QQ** | 使用 QQ 账户登录 |

## 账号信息修改

你可以在个人信息页面中更新以下信息：

- 头像
- 显示名称
- 用户名
- 手机号码
- 登录密码

## 异常登录检测

系统会自动监测异常活动，包括：

- 新的登录地点
- 未识别的设备登录

当检测到可疑活动时，系统会：

1. 向你的注册邮箱发送安全通知。
2. 触发二次验证。
3. 要求你在继续操作前确认身份。

::: warning 安全提醒
如发现任何可疑的登录活动，请立即在[修改密码](/docs/user/edit/)页面更改你的密码。
:::
