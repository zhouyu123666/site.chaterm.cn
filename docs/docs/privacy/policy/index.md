# 隐私政策

Chaterm 收集匿名使用数据以改进产品。你的代码、提示词和对话内容绝不会被收集。

## 概述

Chaterm 使用 [PostHog](https://posthog.com)（一个开源分析平台）收集和分析匿名使用数据。这个反馈循环帮助团队了解开发者如何使用产品，并指导功能和用户体验的改进。

我们的遥测实现是完全透明的 —— 你可以查看[源代码](https://github.com/chaterm/Chaterm)以确切了解我们跟踪的内容。

::: info 透明度
所有收集的数据在离开你的设备之前都经过匿名化处理，不包含任何个人身份信息（PII）。你可以随时在公开仓库中审计遥测实现。
:::

## 我们跟踪什么

我们收集基本的匿名使用数据，包括：

| 类别 | 示例 |
|----------|----------|
| **任务交互** | 任务的开始和结束时间、对话流程（不含内容） |
| **模式与工具使用** | 所选的 AI 模式、调用的工具 |
| **令牌使用量** | 用于成本估算的对话长度指标（不包含令牌的实际内容） |
| **用户界面活动** | 各界面功能的使用情况 |

## 如何退出

遥测功能完全是可选的：

- 当你首次安装或更新 Chaterm 时，会显示关于匿名遥测的通知。
- 你可以随时在隐私设置页面更改偏好。

有关数据收集、脱敏和同步的完整控制，请参阅[隐私设置](/docs/privacy/settings/)。
