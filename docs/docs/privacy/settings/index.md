# 隐私设置

控制 Chaterm 收集哪些数据、如何处理敏感信息以及是否跨设备同步你的设置。

有关具体跟踪内容和原因的详细信息，请参阅[隐私政策](/docs/privacy/policy/)。

## 遥测

遥测功能发送匿名使用数据和错误报告，帮助改进 Chaterm。不会收集任何代码、提示词或对话内容。

| 数据类别 | 包含内容 |
| ------------------- | ---------------------------------------------------- |
| 任务交互 | 任务的开始/结束时间、对话流程（不含内容） |
| 模式与工具使用 | 所选的 AI 模式、调用的工具 |
| 令牌使用量 | 用于成本估算的对话长度指标 |
| 用户界面活动 | 各界面功能的使用情况 |
| 错误报告 | 匿名化的错误信息 |

**开关**：设置 > 隐私 > 遥测 -- **开启**或**关闭**。

::: tip 建议
开启遥测功能可以帮助 Chaterm 团队优先处理功能需求和更快修复问题。所有数据都经过匿名化处理，不会泄露隐私信息。
:::

## 密文脱敏

密文脱敏功能可在 AI 对话框输出中自动遮盖密码、密钥、令牌和其他敏感信息。

**功能说明**：

- 实时识别常见的敏感信息模式
- 在内容显示在屏幕上之前进行遮盖
- 不影响正常使用

**使用场景**：

- 共享屏幕或录制会话时
- 在公共环境中工作时
- 审查可能包含凭证的 AI 输出时

**开关**：设置 > 隐私 > 密文脱敏 -- **开启**或**关闭**。

::: warning 限制
脱敏功能可以覆盖常见的敏感信息模式，但无法保证 100% 检测。在处理高度敏感的信息时，请务必谨慎。
:::

## 数据同步

数据同步功能使你的配置、资产和密钥在所有使用 Chaterm 的设备上保持一致。

**同步的数据**：

| 数据类型 | 示例 |
| -------------------------- | ------------------------------------- |
| 资产 | 主机连接详情、SSH 密钥 |
| 凭证 | 保存的密码和令牌 |
| AI 历史对话 | 历史 AI 对话记录 |

所有同步数据在传输和存储过程中均使用 KMS 加密。

**开关**：设置 > 隐私 > 数据同步 -- **开启**或**关闭**。

::: warning 云端存储
开启数据同步后，你的配置数据会存储在云端。在敏感环境中启用此功能前，请确保你信任 Chaterm 的安全措施。
:::

::: tip 建议
如果你在多台设备上使用 Chaterm，开启数据同步可以避免重复配置，提供一致的使用体验。
:::

## 最佳实践

**数据最小化** -- 仅启用需要的功能，定期检查隐私设置。

**敏感信息** -- 在演示或录制时保持脱敏功能开启。避免在对话中直接粘贴敏感信息，建议使用环境变量或密钥管理器。

**多设备使用** -- 在共享或公共设备上使用 Chaterm 后，及时清理本地数据。定期检查已同步的数据内容。
