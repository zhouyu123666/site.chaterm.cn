安全设置 ​

本文将指导您如何在 Chaterm 中修改您的账户密码。定期更新密码是保护账户安全的重要措施。

操作步骤 ​

点击左侧菜单中的 User 图标，进入个人信息页面

在个人信息页面中，找到并点击"重置密码"按钮

在密码输入框输入新密码

点击"确认"按钮完成密码修改

CAUTION

提示: 密码必须包含至少 8 个字符, 建议使用字母、数字和特殊字符的组合，以提高安全性, 请不要使用曾经使用过的密码。
