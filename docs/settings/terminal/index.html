终端设置 ​

终端设置用于自定义终端的外观和行为，让您打造个性化的终端使用体验。

终端类型 ​

选择终端模拟器类型，不同的终端类型支持不同的功能和特性。

可选类型：

  • xterm（推荐）：现代终端标准，支持丰富的功能和颜色显示
  • vt100：经典终端类型，兼容性最好
  • 其他类型：根据实际需求选择

使用场景：

  • 连接老旧系统时，选择兼容性更好的终端类型
  • 需要特定功能时，选择支持该功能的终端类型
  • 默认情况下，xterm 类型提供最佳体验

字体设置 ​

配置终端显示字体和字体大小，影响终端内容的可读性和显示效果。

配置选项：

  • 字体选择：选择终端使用的字体（如 Monaco、Menlo、Consolas 等等宽字体）
  • 字体大小：设置字体大小（通常范围 10-24px）

主要特性：

  • 支持等宽字体，确保字符对齐
  • 实时预览字体效果
  • 字体大小调整立即生效

使用场景：

  • 根据屏幕分辨率和观看距离调整字体大小
  • 选择易于阅读的字体，减少眼部疲劳
  • 在分屏或多终端场景下，统一字体设置

建议

推荐使用等宽字体（Monospace），确保代码和命令输出对齐整齐。字体大小建议根据屏幕大小设置为 12-14px。

终端回滚量 ​

设置终端历史记录的行数，决定可以向上滚动查看的历史输出内容。

配置说明：

  • 默认值：通常为 1000 行
  • 可调范围：根据系统资源调整，建议 500-10000 行
  • 影响：回滚量越大，占用内存越多

使用场景：

  • 查看大量日志输出时，增加回滚量
  • 需要追溯历史命令输出时
  • 系统资源有限时，适当减少回滚量

注意

回滚量设置过大会占用较多内存，建议根据实际需求合理设置。如果经常需要查看大量历史输出，建议使用终端搜索功能或日志文件。

光标样式 ​

选择终端光标的显示样式，影响光标在终端中的视觉效果。

可选样式：

  • 块状（Block）：光标显示为实心方块，最醒目
  • 下划线（Underline）：光标显示为下划线，占用空间小
  • 竖线（Line/Bar）：光标显示为竖线，类似文本编辑器

使用场景：

  • 块状光标：适合快速定位光标位置
  • 下划线光标：适合需要更多显示空间时
  • 竖线光标：适合精确编辑和定位

触摸板缩放 ​

开启或关闭触摸板手势缩放终端字体功能。

功能说明：

  • 开启：支持使用触摸板手势（如双指捏合）缩放终端字体
  • 关闭：禁用触摸板缩放功能，避免误操作

使用场景：

  • 需要快速调整字体大小时，开启此功能
  • 避免误触缩放时，关闭此功能
  • 在笔记本电脑上使用时，此功能特别有用

SSH Agents ​

配置 SSH 密钥代理，用于管理 SSH 密钥和身份认证。

功能说明：

  • SSH Agent 转发：在 SSH 连接中转发本地 SSH Agent
  • 密钥管理：管理用于 SSH 连接的密钥
  • 自动认证：配置后可以自动使用密钥进行身份认证

使用场景：

  • 需要跳板机连接时，配置 SSH Agent 转发
  • 管理多个 SSH 密钥时
  • 简化 SSH 连接认证流程

建议

SSH Agent 配置需要一定的 SSH 知识，建议参考相关 SSH 配置文档。配置正确后可以显著简化多主机连接流程。

代理配置 ​

配置终端连接使用的网络代理，用于访问受限网络或提高连接速度。

配置选项：

  • 协议类型：选择代理协议（HTTP、HTTPS、SOCKS5 等）
  • 代理地址：代理服务器的主机名或 IP 地址
  • 端口号：代理服务器的端口
  • 身份认证：代理服务器的用户名和密码（如需要）

使用场景：

  • 访问需要代理的网络环境
  • 提高连接速度和稳定性
  • 通过代理访问特定资源

注意

代理配置会影响所有终端连接，请确保代理服务器稳定可靠。配置错误可能导致无法连接。

鼠标事件 ​

配置鼠标中键和右键在终端中的行为。

配置选项：

  • 鼠标中键：选择中键点击的行为（粘贴、无操作等）
  • 鼠标右键：选择右键点击的行为（显示菜单、粘贴等）

主要特性：

  • 支持自定义鼠标事件行为
  • 提供多种事件选项
  • 配置立即生效

使用场景：

  • 习惯使用鼠标中键粘贴时
  • 需要禁用某些鼠标操作时
  • 自定义鼠标交互方式

最佳实践 ​

性能优化 ​

  • 根据实际需求设置终端回滚量，避免占用过多内存
  • 选择合适的字体大小，平衡可读性和显示内容量
  • 合理配置代理，避免影响连接速度

个性化配置 ​

建议

首次使用时，建议：

  1. 根据屏幕大小调整字体和字体大小
  2. 选择舒适的光标样式
  3. 根据使用习惯配置鼠标事件
  4. 在熟悉系统后再配置 SSH Agents 和代理

多终端场景 ​

  • 统一字体设置，保持视觉一致性
  • 根据终端用途调整回滚量（日志查看终端可以设置更大值）
  • 合理使用触摸板缩放，快速调整字体大小
