账号管理 ​

账号管理功能提供完整的账户生命周期管理，包括注册、登录、安全等全方位管理。

账户注册 ​

邮箱注册 ​

使用邮箱地址进行账户注册，支持验证码验证。

第三方认证 ​

国外平台 ​

  • Google - 使用 Google 账户快速登录
  • GitHub - 使用 GitHub 账户快速登录

国内平台 ​

  • QQ - 使用 QQ 账户快速登录

账号信息修改 ​

可以修改以下个人信息：

  • 显示名称
  • 用户名
  • 手机号码
  • 登录密码

异常登录检测 ​

系统会自动检测以下情况：

  • 新的登录地点
  • 新的设备登录

当检测到异常登录时，系统会：

  1. 发送安全通知
  2. 触发二次验证
  3. 要求身份确认

安全提醒

如发现异常登录活动，请立即修改密码。
