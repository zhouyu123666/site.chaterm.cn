文件管理 ​

文件管理功能提供类似 SFTP 的文件浏览和编辑能力，支持在图形界面中管理远程服务器文件。

浏览文件 ​

在左侧菜单点击"文件管理"，系统会显示当前已连接主机的文件树。文件树以树形结构展示服务器的目录和文件，支持：

  • 展开/折叠：点击文件夹可以展开或折叠目录
  • 刷新：右键点击目录可以刷新文件列表
  • 搜索：在文件树中搜索文件或目录

编辑文件 ​

双击文件可以打开文件编辑器。编辑器支持：

  • 语法高亮：根据文件类型自动识别并高亮语法
  • 保存文件：编辑完成后，点击保存按钮将更改保存到远程服务器

编辑器提供类似 Sublime 的编辑体验，支持多语言语法高亮，让您在终端内获得类 IDE 的现代化文件编辑体验。

文件传输 ​

文件管理功能支持文件的上传和下载：

  • 上传文件：可以将本地文件上传到远程服务器
  • 下载文件：可以将远程服务器的文件下载到本地
  • 传输进度：文件传输过程中会显示传输进度和速度

上传操作 ​

  1. 在文件树中选择目标目录
  2. 点击上传按钮或拖拽文件到文件树区域
  3. 选择要上传的文件或文件夹
  4. 等待传输完成

下载操作 ​

  1. 在文件树中选择要下载的文件或文件夹
  2. 右键点击选择"下载"或点击下载按钮
  3. 选择本地保存位置
  4. 等待传输完成

文件管理操作 ​

文件管理支持以下对文件或文件夹的操作：

  • 重命名：修改文件或文件夹名称
  • 权限：修改文件权限（chmod）
  • 复制：文件复制操作
  • 移动：文件移动操作
  • 删除：文件删除操作（请谨慎操作）

操作方式 ​

  • 右键菜单：右键点击文件或文件夹，选择相应操作
  • 快捷键：部分操作支持快捷键（具体请参考快捷键设置）

系统要求 ​

重要提示

文件管理功能依赖于服务器的 SFTP 服务。如果服务器未启用 SFTP 服务，文件管理功能将无法正常使用。

请确保：

  • 服务器已安装并启用 SFTP 服务
  • 用户账户具有相应的文件系统访问权限
  • 网络连接稳定，支持文件传输

注意事项 ​

  • 大文件传输可能需要较长时间，请耐心等待
  • 某些系统文件可能需要管理员权限才能修改
  • 文件传输过程中请勿关闭连接
  • 删除操作不可恢复，请谨慎操作
  • 编辑文件时注意保存，避免数据丢失

最佳实践 ​

文件编辑 ​

  • 编辑重要文件前建议先备份
  • 使用语法高亮功能提高代码可读性
  • 保存前检查文件内容，确保无误

文件传输 ​

  • 大文件传输建议在网络稳定时进行
  • 传输过程中不要关闭连接
  • 定期检查传输进度和状态

权限管理 ​

  • 修改文件权限前了解权限含义
  • 谨慎修改系统文件权限
  • 遵循最小权限原则
