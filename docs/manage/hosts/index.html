主机管理 ​

Hosts 提供统一的主机资产管理功能，支持个人资产和堡垒机资产的管理与维护。

资产管理 ​

资产管理模块用于管理用户自主添加和维护的主机资源。通过左侧导航菜单进入"Hosts"，系统默认显示个人资产列表。

添加主机 ​

添加主机需要完成以下步骤：

  1. 点击界面右上角的"添加"按钮
  2. 在展开的添加主机侧边栏中，填写以下配置信息：
配置项说明必填
个人或企业个人资产选择"个人"，堡垒机资产选择"企业"是
连接 IP 或地址输入目标主机的 IP 地址或域名是
端口SSH 服务端口，默认值为 22是
用户名SSH 登录用户名是
认证方式选择密码认证或密钥认证是
SSH 代理设置如需通过代理服务器连接，可在此配置 SSH 代理否
别名为主机设置便于识别的名称是
分组选择或输入主机所属的群组（如 development、production 等）否

认证方式说明：

  • 密码认证：输入 SSH 登录密码进行身份验证
  • 密钥认证：选择已添加的 SSH 密钥，或先添加密钥后再进行选择
  1. 完成信息填写后，点击"创建"按钮完成主机添加

主机导出 ​

在资产管理界面，点击导出按钮可将主机资产数据导出为 .json 格式文件，保存至本地存储。

主机导入 ​

在资产管理界面，点击导入按钮，选择相应的文件可将主机资产导入至 Chaterm 系统。

支持的导入格式：

  • JSON：Chaterm 标准格式
  • XSH/XTS：XShell 会话文件格式
  • INI/XML：SecureCRT 配置文件格式
  • MXTSESSIONS：MobaXterm 会话文件格式

连接主机 ​

在主机列表中，点击目标主机即可建立 SSH 连接并打开终端窗口。终端窗口提供以下功能：

  • 多标签管理：支持同时打开多个主机连接，通过标签页进行切换
  • 终端分屏：支持水平分屏和垂直分屏，便于同时查看多个终端会话
  • 右键菜单：右键点击终端标签可执行关闭、重命名、克隆等操作
  • 终端搜索：通过快捷键或菜单打开搜索功能，在终端输出中进行内容检索

编辑主机 ​

在资产管理界面，通过主机右键菜单选择"编辑"操作，可修改以下配置信息：

  • 连接 IP 或地址
  • 端口
  • 用户名
  • 认证方式（密码认证或密钥认证）
  • SSH 代理设置（可选）
  • 别名
  • 分组

完成修改后，点击"保存"按钮保存主机配置信息。

克隆主机 ​

在资产管理界面，通过主机右键菜单选择"克隆"操作，系统将创建该主机的副本。克隆后的主机别名将自动添加 _Clone 后缀，同时支持编辑所有配置信息。

完成配置后，点击"创建"按钮完成主机添加。

删除主机 ​

在资产管理界面，通过主机右键菜单选择"删除"操作，系统将弹出确认对话框进行二次确认。确认删除后，该主机的所有信息将被永久删除。

堡垒机资源 ​

组织资产为企业级资产资源，通常通过堡垒机系统同步获取。

添加堡垒机 ​

在工作空间中选择"堡垒机资源"标签页，点击右上角的"资产管理"按钮，填写堡垒机配置信息。在"个人或企业"选项中选择"企业"，其他配置项与添加主机操作相同。

同步堡垒机资产 ​

在堡垒机资源页面，可查看已添加的堡垒机信息列表。点击对应堡垒机的"刷新"按钮可手动同步最新的资产信息。

组织资产的使用方式与个人资产相同，点击资产项即可建立连接。通过右键菜单可执行添加收藏、添加备注或自定义分组等操作。

自定义分组管理 ​

在工作空间中选择"堡垒机资源"标签页，点击右上角的"自定义分组"按钮，可创建自定义分组：

  • 分组名称：输入分组名称
  • 分组描述：输入分组描述（可选）

在堡垒机资源页面，通过自定义分组的右键菜单可执行编辑分组和删除分组操作。

Hosts 管理 ​

搜索功能 ​

在主机列表中，通过搜索框输入 IP 地址或别名进行快速检索，支持实时过滤显示匹配的主机。

收藏功能 ​

在主机列表中，通过主机右键菜单可将主机添加到收藏夹或从收藏夹中移除，便于快速访问常用主机。

最佳实践 ​

合理分组 ​

建议按照项目、环境或用途对主机进行分组管理，以提高管理效率和查找便捷性。常见分组示例：

  • 开发环境
  • 测试环境
  • 生产环境
  • 数据库服务器
  • Web 服务器

安全认证 ​

建议优先使用 SSH 密钥认证方式，以提高连接安全性。密钥管理相关操作请参考 密钥管理 文档。

详细备注 ​

为主机添加清晰的备注信息，记录重要配置参数和使用说明，便于团队成员理解和使用。

定期维护 ​

建议定期检查和更新主机配置信息，确保连接信息的准确性和有效性，及时清理不再使用的主机资源。
