密钥管理 ​

密钥管理用于管理 SSH 密钥，支持使用密钥进行无密码登录。

添加密钥 ​

在左侧菜单点击"密钥管理"，进入密钥管理页面。点击"添加密钥"按钮，打开添加密钥对话框：

填写密钥信息 ​

字段说明必填
名称为密钥设置一个名称是
私钥手动输入私钥内容，或点击下方的"拖拽私钥文件到此处导入"导入本地密钥文件是
公钥手动输入公钥内容否
私钥的密码输入私钥的密码（如果私钥设置了密码）否

导入方式 ​

方式一：文件导入

  • 将私钥文件直接拖拽到指定区域
  • 系统自动识别密钥格式和类型
  • 支持 .pem、.key、.rsa、id_rsa 等格式

方式二：手动输入

  • 在私钥输入框中粘贴完整的私钥内容
  • 格式示例：
    -----BEGIN RSA PRIVATE KEY-----
...
-----END RSA PRIVATE KEY-----

填写完成后，点击"创建"即可添加密钥。

管理密钥 ​

在密钥列表中，可以查看所有已添加的密钥。点击密钥项可以查看详细信息，也可以进行编辑或删除操作。

使用密钥 ​

密钥可以用于：

  • 添加主机时选择密钥认证：在添加主机时，认证方式选择"密钥认证"，然后从下拉列表中选择已添加的密钥
  • 连接主机时自动使用密钥进行身份验证：配置了密钥认证的主机，连接时会自动使用对应的密钥进行身份验证

编辑密钥 ​

  • 点击密钥列表中的"编辑"按钮
  • 修改密钥名称或更新密钥内容
  • 保存更改后立即生效

删除密钥 ​

  • 选择要删除的密钥
  • 确认删除操作
  • 密钥将被永久移除

注意事项

删除密钥前请确保：

  • 该密钥未被其他主机使用
  • 已备份重要的密钥信息
  • 了解删除操作的不可逆性

最佳实践 ​

密钥命名规范 ​

为密钥设置清晰的名称，便于识别和管理：

  • 个人服务器密钥
  • 公司生产环境密钥
  • 开发测试环境密钥

密钥安全 ​

  • 定期更换密钥
  • 不要将私钥内容分享给他人
  • 为私钥设置密码保护
  • 妥善保管私钥文件

密钥备份 ​

建议定期备份重要的密钥文件，避免密钥丢失导致无法连接服务器。
