快速上手 ​

本指南将将带您体验 Chaterm 的核心功能。完成后，您将熟悉 Chaterm 进行常见的终端 Agent 操作。

第一步：安装与登录 ​

安装 Chaterm ​

  1. 下载 Chaterm：
⤓下载 Chaterm
  1. 启动 Chaterm 应用

登录账户 ​

Chaterm 支持多种登录方式：

国内版：

  • 邮箱验证码登录：输入邮箱地址，获取验证码登录
  • 账号密码登录：使用用户名和密码登录
  • 第三方登录：支持 Google、GitHub 登录

国际版：

  • 邮箱验证码登录：输入邮箱地址，获取验证码登录
  • 账号密码登录：使用用户名和密码登录
  • 第三方登录：支持 QQ 登录

提示

  • 首次使用邮箱登录时，如果是新用户将自动注册并登录
  • 如果暂时不想登录，可以点击"跳过"按钮，但未登录状态下将无法使用内置的 AI 模型能力

第二步：添加主机 ​

添加个人主机 ​

  1. 在左侧菜单点击"工作空间"（Hosts）
  2. 点击右上角的"添加"按钮或右下角的"+"按钮
  3. 填写主机信息：
    • 个人或企业：选择"个人"
    • 连接 IP 或地址：输入主机的 IP 地址或域名
    • 端口：SSH 端口，默认为 22
    • 用户名：SSH 登录用户名
    • 认证方式：
      • 密码认证：输入 SSH 登录密码
      • 密钥认证：选择已添加的密钥（需先在密钥管理中添加）
    • SSH 代理设置（可选）：如需通过代理连接可配置
    • 别名：为主机设置便于识别的名称
    • 分组：选择或输入主机所属的群组
  4. 点击"创建"按钮完成添加

连接主机 ​

在主机列表中，点击任意主机即可建立 SSH 连接，打开终端窗口。

第三步：体验 AI 功能 ​

打开 AI 对话面板 ​

有两种方式打开 AI 对话：

  1. 点击左侧菜单的"AI"图标
  2. 在终端中使用快捷键：
    • macOS：Cmd + L
    • Windows/Linux：Ctrl + L

创建新对话 ​

  1. 点击"新建对话"按钮
  2. 选择交互模式：
    • Chat 模式：与 AI 对话，学习，头脑风暴（无法操作服务器）
    • Command 模式：在当前活跃终端执行命令，AI 会规划任务并执行命令
    • Agent 模式：到任意主机执行命令查询，排查错误和任务处理等任何事情
  3. @主机（Agent 模式）：可以 @ 多主机操作
  4. 选择大模型：从模型列表中选择不同的模型
  5. 开始对话

终端内 AI 交互 ​

在终端窗口中，有两种方式快速使用 AI：

  1. 快捷键 Ctrl+K 或 Cmd+K：弹出对话框，输入你想要执行的操作
  2. 快捷键 Ctrl+L 或 Cmd+L：快速打开 AI 对话，AI 会自动获取当前终端的上下文信息

常用场景示例 ​

场景 1：系统监控 ​

@127.0.0.1 查看系统内存使用情况
@server1 检查磁盘空间使用率
@server2 查看当前运行的进程和 CPU 使用率

场景 2：日志分析 ​

@server1 分析最近一小时的错误日志，找出问题原因
@server2 查看应用日志，定位性能瓶颈

场景 3：文件操作 ​

@127.0.0.1 查找最近修改的文件
@server1 备份重要配置文件到 /backup 目录
@server2 清理 7 天前的临时文件

场景 4：服务管理 ​

@server1 检查 Nginx 服务状态，如果异常则重启
@server2 查看 Docker 容器运行情况
@server3 检查数据库连接状态

场景 5：网络诊断 ​

@server1 检查网络连接状态
@server2 测试到特定主机的连通性
@server3 查看端口占用情况

恭喜！ 您已经成功完成了 Chaterm 的快速上手。现在您可以开始使用 AI 驱动的智能终端来提高工作效率了！
